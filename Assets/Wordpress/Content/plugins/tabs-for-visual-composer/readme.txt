== Description ==
<pre>Only and first WPBakery Page Builder Addon of Tabs</pre>
Tabs for WPBakery Page Builder is really simple <a href="http://codecanyon.net/item/visual-composer-page-builder-for-wordpress/242431?ref=labibahmed">WPBakery Page Builder</a> addon that will allow user to put any content in tabs of different style and with different color scheming.You can also display content using shortcode.It contains really simple settings and easy to use.You can set it up in few seconds.You can display Posts,Pages,Portfolios,Testimonials,Gallery and many more.
